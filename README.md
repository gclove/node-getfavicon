node-getfavicon
===============

_node-getfavicon_ is a Node.js module which can extract the favicon url from
a website.

usage
-----
```javascript
const getfavicon = require('getfavicon')

let favicon = getfavicon('http://example.com/path/to/file')
console.log(favicon)
```
